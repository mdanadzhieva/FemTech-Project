Resources

Whether you want to gain a new skill or jump into the world of tech, the below free resources will support you on your journey!


Code First Girls

Code First Girls runs free, part-time coding courses for female/non-binary identifying young individuals across the UK and Ireland. From beginner to more advanced, CFG aim to make breaking into the Tech industry for women super accessible and create the next generation of future Tech leaders.


W3 Schools

W3 Schools is an online school for web developers, covering all the aspects of Web Development, as well as a wide selection of tutorials on several programming languages like Java and Python. Their tutorials start from basic level and move all the way up to professional references.


Free Code Camp

freeCodeCamp is a free, online, self-paced coding bootcamp. There are many paths you can choose: Web Development, Quality Assurance, Machine Learning, Information Security, and more. Their courses are structured around small, digestible amounts of new information.


Code Academy

Codecademy is a user-friendly, varied platform, which makes it a very enticing for budding coders. They have a huge catalogue of free coding classes and paid resources via their Pro collection, which come with certificates of completion.


Upskill

This Skills Bootcamp in Digital Marketing and Web Design is an exciting 4 week programme designed to equip learners with the skills, knowledge and experience they need to secure digital roles. To be eligible for this programme, you must not be in full time employment or full time education and you must be London based.


Skill Crush

Skillcrush is an online coding and design school for career changers. It offers paid and free courses which are project-based and mentor-led, designed to help you break into the Tech world!
